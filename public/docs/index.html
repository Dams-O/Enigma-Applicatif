    Serveur Applicatif

    Fonction onOpen

    Connexion à la base de données pour pouvoir effectuer différentes requêtes. Le serveur se connecte au client, puis envoi les informations au même client afin de tester différentes clés

    Example request:

    curl -X POST \
    "http://localhost:8000/" \
    -H "Content-Type: application/json" \
    -H "Accept: application/json" \

    const url = new URL(
    "http://localhost:8000"
);

let headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};
    .then(response => response.json())
    .then(json => console.log(json));

    Parameter Description
